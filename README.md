# POC Microfrontend using module federation

## Steps to run:

* yarn install
* yarn dev
* access:
    * host http://localhost:3000
    * app-react: http://localhost:3001

## How to create new app

* go to `apps` dir
* run `npx create-mf-app`
* add new package.json script `"dev": "yarn start:live"`
